StoreUI enhances your WooCommerce store management with advanced features and integrations via StoreUI.net.

== Description ==

**Important: This plugin relies on StoreUI.net as a third-party service**

StoreUI plugin connects your WooCommerce store to StoreUI.net services. This connection is established when you activate your license key within the plugin settings. The plugin will then generate and send WooCommerce API keys to StoreUI.net to enable the advanced features and integrations.

For more information about StoreUI services, please visit: [https://storeui.net](https://storeui.net)

To review our Terms of Service, please visit: [https://storeui.net/terms](https://storeui.net/terms)

For our Privacy Policy, please visit: [https://storeui.net/privacy-policy](https://storeui.net/privacy-policy)

Key features:

* Seamless integration with WooCommerce
* Advanced store management capabilities
* Enhanced product management
* Improved order processing
* Custom API endpoints for external integrations

StoreUI is perfect for store owners looking to take their WooCommerce store to the next level.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/storeui` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->StoreUI screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, StoreUI is designed to work with WooCommerce and requires it to be installed and activated.

= Is StoreUI compatible with my theme? =

StoreUI is designed to be compatible with all WordPress themes. However, if you encounter any issues, please contact our support team.
